I am a modal. I use javascript and css only without any dependencies or external libraries like jquery.

You don't need to write any javascript or call any functions to use me: All you need to do is add me to your HTML. (Of course, you will need to include modal.js and modal.css into your HTML document. Make sure to put the script tag for modal.js at the END of your HTML file, so that the HTML has time to load before the funtions in there run.)

Unlike the first modal, I triggered this modal with javascript. (Open the HTML for this page and look for the div with id 'manual-trigger' and notice its onclick event: that's where this modal is getting triggered.)

This could be useful if you want your modal to open after doing some work, for example, after a call to the server (maybe you want to open a modal to display information from the server).

To open a modal via javascript, add this line to your js file: open_modal(document.getElementById("your-modals-id")); (where "your-modals-id" is the id of a div of class "modal", as structured in this page.).

You don't need to worry about closing the modal; modal.js has already handled that.
